## Array (陣列)

### 定義
陣列是一種線性資料結構,將相同類型的元素儲存在連續的記憶體位置中。

### 特性
- **固定大小**:大多數程式語言中陣列大小在宣告時確定
- **隨機存取**:可以透過索引直接存取任何元素
- **記憶體連續**:元素在記憶體中連續排列

### 時間複雜度
- 存取:O(1)
- 搜尋:O(n)
- 插入:O(n) (需要移動元素)
- 刪除:O(n) (需要移動元素)

### 優點
- 快速的隨機存取
- 記憶體效率高(無額外指標開銷)
- 快取友善(連續記憶體)

### 缺點
- 固定大小,不易擴充
- 插入和刪除效率低
- 可能浪費記憶體空間

### 應用場景
- 需要頻繁隨機存取的場景
- 元素數量已知且固定
- 實作其他資料結構的基礎

### 程式碼範例
```python
# 宣告與初始化
arr = [1, 2, 3, 4, 5]

# 存取元素
print(arr[0])  # 1

# 修改元素
arr[2] = 10

# 遍歷
for i in range(len(arr)):
    print(arr[i])
```
